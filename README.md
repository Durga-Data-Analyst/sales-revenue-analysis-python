# Sales & Revenue Analysis using Python

## 📌 Project Objective
This project performs exploratory data analysis (EDA) on sales data to uncover revenue trends, product performance insights, and regional sales patterns.

---

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📊 Analysis Performed

- Monthly Revenue Trend Analysis
- Top Performing Products
- Sales by Region


---

## 🔍 Key Insights

- Revenue shows strong seasonal spikes.
- Top 20% products contribute majority of revenue.
- Certain regions outperform others significantly.
- Repeat customers generate higher revenue share.

---

## 🚀 Skills Demonstrated

- Data Cleaning & Transformation
- Exploratory Data Analysis (EDA)
- Data Visualization
- Business Insight Extraction
